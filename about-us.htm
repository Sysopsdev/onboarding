DivergeIT Agent Install (MacOS)

How to install

1. Download Agent Installation
2. Extract contents by double-clicking the .zip file
3. Open extracted file and run .mpkg file
4. Follow through prompts untill installation is complete
NOTE: Password is required
5. Open settings and naigate to Privacy and Security
6. Navigate to Full Disk access than Enable LTechAgent

If LTechAgent is not visable Please Contact Onboarding Support
